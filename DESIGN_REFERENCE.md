# DESIGN_REFERENCE.md

## 目标画面
目标界面是一个个人作战仪表盘。

结构：
左侧深色导航栏。
右侧浅色主内容区。
顶部是 5 个数据卡片：
今日作战分
任务完成率
运营分
身体分
风险提醒

中间左侧：
今日任务清单，按赚钱、学习、身体、资金、复盘分类。

中间右侧：
风险提醒
今日运营概览
今日身体概览
资金雷达概览

视觉要求：
简洁
清爽
克制
有战斗感
像个人控制台
不要像飞书数据库
不要像企业 BI 大屏
不要堆无意义图表

## 飞书版问题
飞书版已经试过，生成了数据表和仪表盘，但效果太像后台：
表很多
入口散
视觉不统一
仪表盘太空
大面积深色压迫
数据为空时全是 0
手机填写体验一般

## 网页 App 方向
网页 App 要保留飞书版的数据逻辑，但把体验做成真正的个人作战台。
